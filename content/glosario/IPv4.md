**Definición:** La cuarta versión del Protocolo de Internet, que utiliza direcciones de 32 bits para la identificación de dispositivos. La cuarta versión del Protocolo de Internet, utilizada para identificar dispositivos en una red con direcciones de 32 bits. Una dirección de 32 bits utilizada para identificar dispositivos en una red, que consiste en una porción de red y una porción de host. La cuarta versión del Protocolo de Internet, utilizando direcciones de 32 bits para identificar dispositivos en una red.
